
# Dr Wash Laundry

Dr Wash Laundry is a trusted laundry service in Cape Town that provides wash, dry, ironing, and packaging solutions tailored for households, professionals, and businesses. Offering affordable rates, convenient pick-up and delivery, and specialized cleaning for items such as linen, curtains, and shoes, the company is designed for anyone seeking reliable and time-saving laundry care. With a strong community focus and a commitment to quality, Dr Wash makes laundry simple and stress-free.




## Features

- **Responsive Design**: The website is designed to be mobile-friendly and adjusts to different screen sizes.
- **Navigation Bar**: A consistent navigation bar is included on all pages for easy access.
- **Footer**: Each page includes a footer with social media links, contact details, and the address.

## Changelog(Part 3)
**New Features & Interactive Elements
- Services Accordion: Implemented an interactive accordion on services.html. Clicking a service header now smoothly toggles the visibility of the corresponding content.
- Pricing Lightbox Gallery: The pricing page view was changed to a gallery structure with a lightbox modal for viewing pricing list images in a larger format.

Structural & Code Changes
- File Structure Refinement: Moved the original contact content to a dedicated enquiry.html page.

- JavaScript Consolidation: Created and linked script.js to manage all dynamic elements (Accordion, Lightbox, Modal).

- Service Page Structure: Modified services.html to separate each service into distinct <div> structures to support the accordion effect.

Contact & Location Improvements
- Clickable Contact Info:
- Made the Physical Address on contact.html a clickable link to Google Maps.

- Made Phone Numbers clickable using tel: links.

- Made the Email Address clickable using a mailto: link.

- Static Map: Added a static map image to the contact.html page for easy location visualization.

 Responsiveness (CSS)
- Footer Layout Fix: Added necessary CSS rules (using flexbox and word-wrap) to the footer to ensure text, links, and contact details do not cut off on mobile devices, maintaining a fully visible structure.
